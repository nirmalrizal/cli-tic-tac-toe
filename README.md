
## Setup project locally
1. Clone the project
  ``` git clone https://github.com/nirmalrizal/cli-tic-tac-toe ```
2. Navigate to project folder 
   ``` cd cli-tic-tac-toe ```
3. Install dependencies
  ``` npm install ``` or ``` yarn ```

## Start server

To start the server type

``` node server ```

## Start Client

To start the client type

``` node client ```
